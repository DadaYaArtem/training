Для инициализации репозитория - пишем git init .

Чтобы понять текущее состояние репозитория - пишем git status

Чтобы добавить файл в staging area - пишем git add имя файла
(git add -u добавляет в area все файлы из контроля git которые изменились)

Чтобы сделать комит - пишем git commit -m "About commit"

Чтобы посмотреть историю коммитов - пишем git log

Чтобы перейти к конкретному комиту - пишем git checkout циферки из git log который нам нужен



Чтобы связать удаленный репозиторий с локальным один раз пишем - git remote add origin адресс_репозитория

Чтобы глянуть список удаленных репозиториев - пишем git remote --v

Чтобы залить код на удаленный репозиторий - пишем git push origin master

Чтобы подтянуть код с удаленного репозитория - пишем git pull orogin master
